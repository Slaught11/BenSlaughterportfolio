"Sonic Racing Trials:"

"Sonic Racing Trials" Is a first person driving game based on the Netflix series "Sonic Prime".

This project was created by myself and Joe Vine, a fellow digital media design student.

Joe took the lead on this project, creating the gamification elements and laying out a wireframe for the racetrack, while I focused on creating the game menu screens, 3D models and prototyping.

The project entails a difficult challenge-based racetrack map where the player has to finish the course in a certain time or they will be eliminated. The course contains a range of obstacles and perks that allow for a dynamic gameplay experience.

"Architecture"

My recent project "Architecture", an immersive puzzle game where the player is tasked with searching each complex level to find the Architect's sketchbook, all while facing different obstacles and problems along the way.

This project was created primarily through Unreal Engine 5 game development, alongside additional production from 3D modelling software Blender and Unity.
