"Inked Expressions":

The "Inked Expressions" Prototype website is a digital piece I worked on towards the end of my second academic year. The project was my first attempt at website design and I wanted to create a minimalistic, yet interactive site with potential for user playability.

The project consists of a coded landing page with additional menu pages with different features on each page.

The main function of the website is the tattoo generator section, which allows the user to randomly generate a different tattoo from an image database at the click of a button.

"ES Studios Prototype Landing Page":

This piece is a conceptual web design focused project which I undertook after graduating from Plymouth University.

The "ES Studios Prototype Landing Page" is a conceptual design for my sister Evie Slaughter to showcase her film studies work.

The piece will be incorporated into a full scale, portfolio-style website in the future to reflect an academic career in Film Studies.
